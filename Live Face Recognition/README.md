# Live face recognition

## Pre requisites
0. Ensure the python `3.11.9`
```
python -V
```
1. Install all the needed libraries
```
pip install -r requirements.txt
```

## Test 
```
python test.py
```

## Main
1. Replace the `./monastry.jpg` image to whatever desired.
2. Execute script
```
python main.py
```
